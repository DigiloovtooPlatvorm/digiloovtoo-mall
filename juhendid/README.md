# Juhendid

Selles kaustas asuvad digiloovtöö malli kasutamiseks vajalikud juhendid.

## Selle kausta failid

- `juhend-õpilasele.md` – juhend õpilasele digiloovtöö repositooriumi kasutamiseks
- `juhend-juhendajale.md` – juhend õpetajale või juhendajale töö jälgimiseks ja juhendamiseks

## Kuidas kasutada

- Kui oled õpilane, alusta failist `juhend-õpilasele.md`.
- Kui oled õpetaja või juhendaja, alusta failist `juhend-juhendajale.md`.
